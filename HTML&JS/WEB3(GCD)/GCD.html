解題方法2: 輾轉相減法

流程圖:

虛擬碼:

1.宣告兩數m,n
2.使用輾轉相減法
3.return m即為GCD

source codes

static int gcd_loop1(int m, int n){

	while (m!=n){

	while (m>n)  m=m-n;

	while (m<n)  n=n-m;

	}

	return (m);

	}

解題方法3: 輾轉相除法 : loop (流程圖、虛擬碼)

流程圖:

虛擬碼:

1. Divide x by y and compute the remainder; call that remainder r.
2. If r is zero, the procedure is complete, and the answer is y.
3. If r is not zero, set x equal to the old value of y, set y equal to r, and repeat the entire

source codes

static int gcd_loop2(int m, int n){

	int t;

	if (m<n){

		t=m;m=n;n=t;}

	while (n!=0){

		t=m%n; m=n; n=t;

	}

	return (m);

}

流程圖:

虛擬碼:
